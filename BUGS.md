# Баг-репорты микросервиса

## ID: БР1
### Заголовок: Ошибка в структуре ответа при создании объявления в ручке POST /api/1/item

### Приоритет: High

### Шаги воспроизведения:
1. Клонировать репозиторий: git clone https://github.com/katerinadoroshenkospb/Avito
2. Перейти в клонированный репозиторий, в подпапку с файлом gradlew.bat
3. Для запуска АТ использовать команду gradlew test
4. Будет сформирован отчёт о запуске АТ. Для его просмотра откройте файл build/reports/tests/test/index.html в браузере.

### ОР: Код и статус ответа 200 ОК, ошибок в структуре ответа нет, объявление создано
{
  "id": "<string>",
  "sellerId": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {
    "likes": "<integer>",
    "viewCount": "<integer>",
    "contacts": "<integer>"
  },
  "createdAt": "<string>"
}
### ФР: Код и статус ответа 200 ОК, ошибка в структуре ответа, объявление создано
{
    "status": "Сохранили объявление - 7a728a98-ac52-48a5-980d-06988169e591"
}
